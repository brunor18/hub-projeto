<h1>Hub de projetos escolares</h1>

Este programa foi baseado em um exercício da faculdade

Recomenda-se executar o programa pelo executável ou pelo terminal padrao do windows, pois o buffer do terminal interno do VSCode ou VS pode quebrar o projeto


Acesse o executável do projeto pelo caminho:

```bash
bin/Debug/net10.0/struct-bruno.exe
```
Ou
```bash
bin/Release/net10.0/struct-bruno.exe
```
